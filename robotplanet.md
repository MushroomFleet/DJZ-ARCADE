## ğŸ“– How to Play

### Controls
- **WASD** or **Arrow Keys** - Move your unit
- **Mouse** - Navigate menus
- **Weapons** - Auto-fire automatically

### Gameplay Loop
1. **Choose Your Unit** - Select from unlocked robot units, each with different stats
2. **Select Game Mode** - Pick from Standard, Hyper, Hurry, and more
3. **Survive the Waves** - Fight off endless robotic enemies
4. **Collect XP** - Defeat enemies to gain experience
5. **Level Up** - Choose weapon or passive upgrades
6. **Unlock Content** - Complete challenges to unlock new units and modes
7. **Meta Progression** - Spend scrap on permanent upgrades between runs

### Objective
Survive as long as possible against increasingly difficult waves of enemies. Reach level 99 or survive the final boss rush to achieve victory!

## ğŸ› ï¸ Development

### Tech Stack
- **React 19** - UI framework
- **TypeScript** - Type-safe development
- **Three.js** - 3D graphics engine
- **React Three Fiber** - React renderer for Three.js
- **Vite** - Build tool and dev server
- **Tauri** - Desktop app framework
- **Zustand** - State management
- **TailwindCSS** - Styling

### Prerequisites
- Node.js (v18 or higher)
- npm, yarn, or pnpm

### Local Development

1. Clone the repository
```bash
git clone https://github.com/MushroomFleet/DJZ-ARCADE.git
cd ROBOT-PLANET
```

2. Install dependencies
```bash
npm install
```

3. Run development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```

### Tauri Desktop Build

1. Run Tauri development mode
```bash
npm run tauri:dev
```

2. Build desktop application
```bash
npm run tauri:build
```
