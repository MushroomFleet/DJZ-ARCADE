## üì• Installation

### Download Options

All games are available in the [Releases](https://github.com/MushroomFleet/DJZ-ARCADE/releases) section as:
- **MSI Installers** - Full installation packages for Windows
- **Portable EXE** - Standalone executables requiring no installation

### Installation Steps

#### MSI Installation (Recommended)
1. Navigate to [Releases](https://github.com/MushroomFleet/DJZ-ARCADE/releases)
2. Download the `.msi` file for your desired game
3. Run the installer and follow the on-screen instructions
4. Launch the game from your Start Menu or Desktop shortcut

#### Portable Installation
1. Download the portable `.exe` file from [Releases](https://github.com/MushroomFleet/DJZ-ARCADE/releases)
2. Save to your preferred location
3. Double-click to run (no installation required)
