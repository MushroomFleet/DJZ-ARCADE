# REACT-CYCLES-ONE

A Tron-inspired light cycle game built with React and Three.js. Navigate your light cycle through a neon-lit grid arena while avoiding trails, obstacles, and walls. Outmaneuver your AI opponent in this fast-paced arcade experience with multiple game modes and a custom level editor.

## üéÆ Game Features

- **Classic Tron Gameplay**: Control your light cycle as it leaves a trail behind, avoiding crashes while trying to make your opponent crash
- **Multiple AI Modes**: Choose from different AI difficulty levels (Classic, Advanced)
- **Dynamic Maps**: Play on various pre-built maps with different obstacle configurations or create your own
- **Level Editor**: Design and save custom maps with an intuitive grid-based editor
- **Replay System**: Watch instant replays of your matches from multiple camera angles
- **Customizable Settings**: Adjust FOV, camera height, grid size, and more
- **Retro Aesthetics**: Neon visuals with a cyberpunk-inspired interface

## üïπÔ∏è Controls

### Gameplay
- **Arrow Left** or **A** - Turn left
- **Arrow Right** or **D** - Turn right
- **Space** or **Enter** - Start game (from menu/game over screen)
- **Escape** - Exit replay

### Level Editor
- **Click** - Toggle obstacles on/off
- **Click + Drag** - Add/remove obstacles in rectangular areas

## üì• Installation

### Option 1: Desktop Application (Windows)

Download and install the latest `.msi` installer from the [Releases](https://github.com/MushroomFleet/REACT-CYCLES-ONE/releases) page.

1. Navigate to the Releases page
2. Download the latest `.msi` file
3. Run the installer
4. Launch REACT-CYCLES-ONE from your Start Menu

### Option 2: Play in Browser

Play the game directly in your browser without installation:

**[üéÆ Play Now](https://scuffedepoch.com/react-cycles-one/)**

### Option 3: Run Locally (Development)

**Prerequisites:** Node.js

1. Clone the repository:
   ```bash
   git clone https://github.com/MushroomFleet/REACT-CYCLES-ONE.git
   cd REACT-CYCLES-ONE
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to the local server address (typically `http://localhost:5173`)
