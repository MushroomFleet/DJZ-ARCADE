# REACT-WARS

> A tactical squad-based strategy game built with React and Three.js

**REACT-WARS** is an immersive 3D tactical combat game where you command elite squads through procedurally generated urban battlefields. Engage in intense firefights, manage strategic resources, and complete high-stakes missions against zealot forces.

### Gameplay Mechanics

#### Squad Management
- Switch between agents to utilize their unique abilities
- Position your squad tactically using cover and elevation
- Coordinate attacks to overwhelm enemy positions
- Manage ammunition and health across your team

#### Combat System
- Each weapon has unique characteristics (damage, range, rate of fire)
- Enemies use AI tactics including cover usage and flanking
- Civilians populate the battlefield - avoid collateral damage
- Destructible environment allows for dynamic tactical opportunities

#### Mission Objectives
- **Sector 7 Infiltration**: Breach enemy defenses and neutralize zealot forces
- **Operation Blackout**: Extract vital data and reach the evac zone
- Complete primary objectives while managing secondary goals

#### Strategic Planning
- Upgrade agent abilities between missions
- Research new technologies to unlock better equipment
- Plan your loadout based on mission requirements
- Manage resources earned from successful operations

## ğŸ› ï¸ Technical Details

### Built With
- **React 19** - UI framework
- **TypeScript** - Type-safe development
- **Three.js** - 3D graphics rendering
- **Vite** - Build tooling
- **Zustand** - State management
- **Tauri** - Desktop application framework

### System Requirements
- Modern web browser (Chrome, Firefox, Edge)
- WebGL 2.0 support
- Recommended: 8GB RAM, dedicated GPU

### Architecture
- **Component-Based**: Modular React architecture
- **ECS Pattern**: Entity-Component-System for game logic
- **Procedural Generation**: Seeded random for consistent level design
- **Performance Optimized**: LOD system, object pooling, and spatial hashing
